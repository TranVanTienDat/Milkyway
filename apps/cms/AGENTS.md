# 📦 MILKYWAY-CMS — Trụ cột Nội dung & API

> **Stack:** Strapi v5 | TypeScript | SQLite.

## 1. Thông số Kỹ thuật
- **Framework:** Strapi v5.
- **Runtime:** Node.js >=20.
- **Cấu hình API:** Mọi endpoint và logic nghiệp vụ nằm tại `src/api/`.

## 2. Chỉ dẫn cho Agent
- Thiết kế Content Type trong Admin UI trước khi can thiệp code.
- Logic nghiệp vụ tập trung tại `Services` của folder `src/api/`.
- Kiểm tra quyền truy cập tại plugin `users-permissions`.
