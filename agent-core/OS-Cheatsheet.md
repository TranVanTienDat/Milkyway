## 1. Lệnh đánh thức (Wake-up)
Luôn chạy lệnh này đầu tiên khi bắt đầu phiên làm việc mới:
- `/wake-up`: Nạp tri thức dự án, Rules và kích hoạt tư duy hệ thống.

## 2. Phím tắt & Kỹ thuật bổ trợ (Shortcuts)
- `Cmd + L`: Mở khung chat với Agent.
- `Cmd + I`: Trình soạn thảo inline (Agent thực thi trực tiếp tại dòng code/văn bản).
- `@`: Gắn thẻ file/folder để nạp ngữ cảnh (Context Injection).
- `#`: Gắn thẻ symbols (Function, Class) để AI tập trung phân tích.

## 3. Quy trình thực thi chuẩn (Universal Workflow)
Giao dịch với AI thông qua 6 bước "vàng":
1. **Discovery**: Nạp ngữ cảnh (`@`, `search_web`).
2. **Design**: Phác thảo giải pháp (kiến trúc, logic).
3. **Planning**: Lập Step-by-Step Plan (**User Approve**).
4. **Implementation**: AI thực thi viết code/content.
5. **Verification**: Chạy thử, đối soát requirement.
6. **Retro**: Đúc kết bài học & Cập nhật Memory (`40-Memory/`).

## 4. Các Slash Commands phổ biến
- `/opsx-new`: Tạo task mới (Development).
- `/opsx-apply`: Thực thi task.
- `/opsx-verify`: Kiểm tra chất lượng.
- `/brainstorming`: Kích hoạt chế độ tư duy sáng tạo/phân tích sâu.
- `/code-reviewer`: Review code theo tiêu chuẩn dự án.

## 5. Thư viện tri thức (Dewey Decimal Index)
- `docs/010-...`: Planning & Sprints.
- `docs/020-...`: Requirements (PRD/Story).
- `docs/030-...`: Technical Specs.
- `knowledge-base/10-Technical/`: Coding Standards.
- `knowledge-base/40-Memory/`: Lessons Learned.
