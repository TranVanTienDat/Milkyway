## ADDED Requirements

### Requirement: Hero Banner Section
Trang chủ cần hiển thị phần Hero Banner chứa ảnh sản phẩm đẹp, slogan thương hiệu và nút "Khám phá ngay" để điều hướng người dùng.

#### Scenario: User arrives at the homepage
- **WHEN** user loads the homepage
- **THEN** they see the main Hero Banner with background image, clear slogan, and an interactive CTA button

### Requirement: Unique Selling Points (USP)
Trang chủ phải nêu rõ 4 giá trị đặc trưng của thương hiệu (Dựa trên khoa học, Làm đẹp an toàn, Thuần chay, Thành phần tự nhiên).

#### Scenario: User scrolls down to USP section
- **WHEN** user views the USP section
- **THEN** 4 icons and corresponding short text labels are displayed clearly, appearing smoothly

### Requirement: Featured Products Grid
Khách hàng có thể nhìn thấy các sản phẩm tiêu biểu ngay tại trang chủ (Serum, Tẩy trang, Mặt nạ, Kem dưỡng) cùng thông tin giá tham khảo.

#### Scenario: User checks featured products
- **WHEN** user scrolls to the Featured Products section
- **THEN** product cards display images, titles, and prices, and a "Xem tất cả" button is accessible

### Requirement: AI Skincare Advisor Integration
Giao diện phân tích da bằng AI cần thu hút sự chú ý, có điểm nhấn UI rõ ràng và mô phỏng được quá trình kiểm tra da 30 giây.

#### Scenario: User interacts with the AI Skincare Advisor
- **WHEN** user clicks the "Kiểm tra chất lượng da" button
- **THEN** the UI provides visual feedback (e.g., loading/analyzing state) before showing a result state

### Requirement: Customer Testimonials Component
Đánh giá từ khách hàng tích cực cần được hiển thị trung thực để làm social proof tăng độ tin cậy.

#### Scenario: User views social proof
- **WHEN** user looks at the testimonials area
- **THEN** they see user names, star ratings, and review texts structured consistently

### Requirement: Press & Brand Logos
Cần có một section footer-like (hoặc pre-footer) hiển thị nổi bật logo các báo/ tạp chí lớn nhắc đến thương hiệu.

#### Scenario: User confirms brand authority
- **WHEN** user views the bottom section of the page content
- **THEN** logos of magazines such as Vogue, Elle, Harper's Bazaar are displayed in a row
